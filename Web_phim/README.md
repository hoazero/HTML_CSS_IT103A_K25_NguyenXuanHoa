# Simple Movie UI

Một trang tĩnh mẫu để xem phim (HTML/CSS/JS). Mở file `index.html` trên trình duyệt để thử.

Hướng dẫn nhanh:

1. Mở file [index.html](index.html) trong trình duyệt (kéo-thả hoặc `Open File`).
2. Chọn phim trong danh sách bên phải để bắt đầu.

Gợi ý: để chạy qua HTTP (nếu trình duyệt chặn một số nguồn), có thể dùng một server tĩnh như `npx http-server` hoặc `python -m http.server`.
